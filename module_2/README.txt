Основной задачей проекта является овладение набором инструментов для выполнения разведывательного анализа данных (Exploratory Data Analysis, EDA).
Основными целями и задачами проекта являются: создание и проверка гипотез о связи различных переменных с результатами госэкзамена по математике для каждого ученика школы, проверка качества данных и их очистка, определение параметров модели, а также отслеживание влияния условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, находящихся в группе риска.
Основными этапами работы являются: загрузка модулей, создание DataFrame на основе имеющихся данных из файла, написание функций для анализа и прдобработки данных, анализ данных по каждому признаку в отдельности, обработка пустых значений, ошибок и выбросов, корреляционный анализ количественных переменных, анализ категориальных признаков, отбор качественных признаков, проверка результатов работы и подведение итогов.
Проект не является командной работой, это самостоятельнрое задание. Мною изучены базовые методы разведывательного анализа данных и составления отчётов по его результатам. И предстоит огромная работа по закреплению полученных знаний и наработке практических навыков.Особый интерес вызвали возможности модулей matplotlib и seaborn при анализе данных а также использование теста Стьюдента для анализа влияния признаков на целевую переменную.
В процессе выполнения работы мною получены лишь базовые навыки предобработки и визуализации данных, а также корреляционного анализа переменных. Для дальнейшего погружения в специальность обязательно дополнительное изучение материалов по теме проекта.
