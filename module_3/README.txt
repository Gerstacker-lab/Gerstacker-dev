Основной задачей проекта является овладение набором инструментов для качественной подготовки данных, предшествующей обучению модели.
Основными целями и задачами проекта являются: тщательная очистка данных и генерация новых признаков (Feature Engineering) способных повысить точность модели, а также построение модели, которая предсказывает рейтинг ресторана.
Основными этапами работы являются: разделение датафрейма на тренировочный и тестовый, тщательная очистка данных и генерация новых признаков, создание, обучение и тестирование модели, используя метрику (показатель), которая называется Mean Absolute Error, и представляет собой среднее абсолютное значение отклонения предсказанных значений от фактических, проверка результатов работы и подведение итогов.
Проект не является командной работой, это самостоятельнрое задание. Мною изучены и закреплены базовые методы тщательной очистки данных и генерации новых признаков с целью повышения точности прогнозов, которые делает модель. Также предстоит огромная работа по закреплению полученных знаний и наработке практических навыков. Особый интерес вызвали возможности модулей datetime и re при анализе и обработке данных.
Для дальнейшего погружения в специальность обязательно дополнительное изучение материалов по теме проекта.
