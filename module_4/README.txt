Основной задачей проекта является овладение набором инструментов для улучшения предсказательной силы модели.
Основными целями и задачами проекта являются: получение новых значений ROC AUC для новых моделей и улучшение их результата на основе подбора параметров регуляризации.
Основными этапами работы являются: разделение датафрейма на тренировочный и тестовый, тщательная очистка данных и генерация новых признаков, использование визуализации, а также очистка выбросов, обучение и тестирование модели, подбор параметров регуляризации для улучшения предсказательной силы нашей модели, проверка результатов работы и подведение итогов.
Проект не является командной работой, это самостоятельнрое задание. Мною изучены и закреплены базовые методы поиска оптимальных параметров для оптимизации работы логистической регрессии.
