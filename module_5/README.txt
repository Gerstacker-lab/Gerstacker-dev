Основной задачей проекта является создание модели, которая будет предсказывать стоимость автомобиля по его характеристикам для того, чтобы выявлять выгодные предложения (покупать ниже рыночной стоимости, продавать выше). В результате парсинга данных с сайта auto.ru результат оказался неудовлетворительным. С учетом отсутствия достаточного количества времени принял решение воспользоваться набором данных с Kaggle.
Основными целями и задачами проекта являются: получение новых значений MAPE (Mean Percentage Absolute Error) для новых моделей и улучшение их результата достижением минимального процента ошибки.
Основными этапами работы являются: разделение датафрейма на тренировочный и тестовый, тщательная очистка данных и генерация новых признаков, использование визуализации, а также очистка выбросов, обучение и тестирование модели для улучшения предсказательной силы нашей модели, проверка результатов работы и подведение итогов.
Проект не является командной работой, это самостоятельнрое задание.
